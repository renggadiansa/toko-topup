# Website Top UP
Website Top UP merupakan aplikasi web toko online sederhana dengan demo produk penjualan diamond dan uc PUBG


##Fitur
- Login
- Register
- Keranjang belanja
- Checkout
- Sistem kupon Belanja
- Pembayaran melalui transfer bank
- Konfirmasi pembayaran
- Formulir kontak

##Hak akses
- Admin
- Customer
